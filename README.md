# 🌦️ Weather App

A simple and responsive Weather App built using Flutter that displays real-time weather data and forecasts using an external API.

## 🚀 Features

- 🌡️ Current temperature display
- ☁️ Weather condition (Cloudy, Rainy, Sunny, etc.)
- 🕒 Hourly weather forecast
- 💧 Additional info (Humidity, Wind Speed, Pressure)
- 🔄 Refresh weather data

## 🛠️ Tech Stack

- Flutter (UI Framework)
- Dart (Programming Language)
- OpenWeatherMap API (Weather Data)
- HTTP package (API calls)
- Intl package (Date formatting)

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/weather-app.git
   ```
2. Navigate to the project folder:
```bash
   cd weather-app
   ```

3. Install dependencies:
```bash
   flutter pub get
   ```

4. Create a `.env` file and add your API key:
```bash
   API_KEY=your_openweathermap_api_key
   ```

5. Run the app:
```bash
   flutter run
   ```
